## WebWechat

基于`Golang`语言和`Gin`框架的个人微信系统

## 使用方式

```shell
# 下载代码
git clone https://xxxx.xxx.xx/xx.git
# 更新依赖
go mod download
# 编译
go build main.go
# 清理无用mod引用
go mod tidy
```

## Docker一键运行

```shell
docker compose up -d
# 或者
docker-compose up -d
```
